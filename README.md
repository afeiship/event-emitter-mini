# event-emitter-mini
> A mini and light event emitter

## apis:
| name | args          | description         |
|------|---------------|---------------------|
| on   | name,handler  | register an event   |
| off  | name,handler  | unregister an event |
| emit | name,arg1,ar2 | fire an event       |
